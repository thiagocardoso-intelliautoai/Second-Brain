# Second Brain - Thiago Cardoso

Este vault e um sistema pessoal de conhecimento em Markdown, local-first, feito para funcionar em Obsidian, Claude, Codex e qualquer IA futura que consiga ler arquivos locais.

Ele nao e uma base corporativa nem um template generico de produtividade. Ele existe para organizar como Thiago pensa, trabalha, escreve, estuda, cria frameworks, documenta processos e transforma conhecimento em entregaveis reais.

O AIOX esta instalado na raiz como camada opcional de orquestracao para Codex e Claude Code. A fonte de verdade operacional continua sendo `AGENTS.md` e o AI Operating System do vault. Para Claude Code, a entrada raiz e `CLAUDE.md`.

Principios centrais:

- Tudo em Markdown simples.
- Raw imports ficam separados de conhecimento destilado.
- Proveniencia importa: notas migradas devem apontar a fonte.
- Conteudo com voz propria nao deve ser reescrito silenciosamente.
- O sistema deve crescer por uso real, nao por taxonomia bonita.
- A IA deve desafiar pensamento fraco, vago ou superengenheirado.
