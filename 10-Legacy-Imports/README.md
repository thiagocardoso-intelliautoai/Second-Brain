# Legacy Imports

Esta pasta guarda exports antigos e material bruto migrado de sistemas que nao sao mais o lugar ativo de organizacao do Second Brain.

## Papel

Use esta pasta apenas como fonte historica e proveniencia.

Ela nao e:

- sistema ativo de captura;
- inbox;
- backlog operacional;
- fonte principal para a IA decidir prioridade.

## Regra

Quando uma parte de um export legado virar conhecimento reutilizavel, crie ou atualize uma nota destilada na pasta de dominio correta e aponte a origem.

Nao reprocessar exports inteiros sem uma tarefa viva.
