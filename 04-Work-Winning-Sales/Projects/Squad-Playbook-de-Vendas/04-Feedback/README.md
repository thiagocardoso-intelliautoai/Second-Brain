# Feedback - Squad Playbook de Vendas

Use esta pasta para feedback que muda a proxima iteracao do Playbook, criterio de qualidade, criterio de pronto ou decisao operacional.

Referencial global:

- [[07-Frameworks-and-Processes/Second-Brain-System/Feedback-Loops-and-Quality-Gates|Feedback Loops and Quality Gates]]

## Quando registrar

Registre aqui quando houver:

- QA relevante de templates, agents, tasks ou outputs;
- feedback de Rafael, consultor usuario ou cliente;
- resultado de piloto ou demonstracao;
- falha que revele problema de processo, prompt, template ou automacao;
- metrica ou evidencia que mude a proxima acao.

## Quando nao registrar

Nao crie nota para:

- microajuste que ja foi resolvido no runtime novo `07-Runtime-Squad/`;
- comentario que nao muda criterio ou proxima iteracao;
- detalhe visual isolado sem decisao;
- duplicacao de log tecnico que ja vive no runtime canonico.
