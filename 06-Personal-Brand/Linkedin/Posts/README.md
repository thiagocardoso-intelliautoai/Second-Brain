# Posts

Cada post em criacao vive em uma pasta propria.

Formato recomendado:

```text
Posts/<slug>/
  POST.md
  PREVIEW.md
  pesquisa.md        opcional
  visual/brief.md    opcional
```

## Regras

- Nao criar `Posts-Publicados` por padrao.
- Se o post foi agendado no LinkedIn, nao precisa registrar.
- Se Thiago quiser historico depois, atualizar o `status` no proprio `POST.md`.
- Preservar original sempre que o post nascer de ideia bruta.
