# Captura Rapida - Ideias LinkedIn

Use este arquivo quando a ideia precisa entrar rapido e ainda nao merece pasta propria.

## Como Capturar

Cole a ideia sem polir.

Formato simples:

```markdown
## YYYY-MM-DD - titulo curto

Original:

-

Possivel tese:

-
```

## Regra De Processamento

Quando Thiago pedir para organizar:

1. preservar o original;
2. separar ideias misturadas;
3. criar pasta propria para as melhores;
4. mover para `Posts/<slug>/` somente quando houver intencao de escrever;
5. arquivar ou manter aqui ideias fracas, sem fabricar importancia.
