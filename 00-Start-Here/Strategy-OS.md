Este arquivo e a camada leve de estrategia do Second Brain.

> O que importa agora e por que?

## Como usar este arquivo

Leia este arquivo antes de trabalho substancial que envolva:

- estrategia;
- projetos ativos;
- melhoria do vault;
- priorizacao;
- decisoes de foco;
- criacao de novos projetos;
- mudanca relevante de contexto.

Nao leia este arquivo como identidade fixa de Thiago. A secao `Snapshot estrategico atual` e uma foto datada.

Regra:

- se o snapshot tiver mais de 30 dias, trate prioridades como possivelmente desatualizadas;
- antes de tomar decisoes de foco com snapshot antigo, pergunte a Thiago o que mudou;
- nao invente destino, ambicao, prazo, bloqueio ou criterio de pronto;
- quando houver conflito entre projeto urgente e direcao estrategica, explicite o trade-off.
- nao dependa de ritual semanal ou mensal manual para manter este arquivo vivo.

## Atualizacao sem ritual

Thiago provavelmente nao vai parar toda semana ou todo mes para preencher revisao. Portanto, este Strategy OS deve ser atualizado por gatilhos naturais, durante o trabalho real com IA.

### Quando atualizar

Atualize este arquivo somente quando uma conversa ou trabalho revelar mudanca real em:

- prioridade;
- projeto importante;
- fonte de renda;
- oferta futura;
- trade-off;
- decisao estrategica;
- criterio de foco;
- risco relevante.

Exemplos de gatilhos:

- Thiago disser que um projeto da Winning ficou urgente ou perdeu importancia.
- Uma mentoria gerar evidencia nova sobre oferta, publico, promessa ou metodo.
- Thiago decidir pausar, acelerar ou arquivar uma frente.
- Um projeto revelar que a prioridade atual nao combina mais com o snapshot.
- O snapshot tiver mais de 30 dias e a IA precisar tomar uma decisao de foco.

### Como atualizar

- Se a mudanca for clara, atualize a secao afetada e registre o que mudou.
- Se a mudanca for ambigua, faca uma pergunta curta antes de editar.
- Se nada estrategico mudou, nao mexa neste arquivo.
- Nao crie logs longos por manutencao estetica.
- Nao cole templates vazios.

### Pergunta minima quando o snapshot estiver velho

Use esta pergunta antes de decisoes de foco com snapshot antigo:

```text
Desde o snapshot de estrategia, algo mudou nas prioridades principais ou Winning Sales continua sendo prioridade 1?
```
