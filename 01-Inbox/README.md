# Inbox

Use esta pasta para captura rapida quando ainda nao sabe onde uma ideia pertence.

O Inbox nao e lugar permanente.

## Como Usar

Jogue aqui:

- ideias soltas;
- links;
- transcricoes;
- prints;
- prompts;
- perguntas;
- notas de livro;
- materiais sem destino claro.

Para captura rapida pelo celular ou Obsidian, use:

`01-Inbox/00-Captura-Rapida.md`

Depois, quando estiver no computador, peca para a IA:

```text
Organiza o Inbox.
```

A IA deve ler `AGENTS.md`, preservar o original e mover cada captura para o destino certo.

## Destinos Provaveis

- projeto ativo;
- ideia de LinkedIn;
- post em criacao;
- referencia;
- framework;
- cliente ou mentoria;
- aprendizado;
- arquivo.

## Regra

Se a captura for conteudo com voz, ideia de post ou frase pessoal, nao reescrever silenciosamente.
