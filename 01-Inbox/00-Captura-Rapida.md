# Captura Rapida - Inbox Geral

Use este arquivo para ideias, insights, perguntas, links e notas soltas que precisam entrar rapido no Second Brain e ainda nao tem destino claro.

Este arquivo e para captura bruta. Nao precisa organizar na hora.

## Como Capturar

Cole abaixo com o minimo de atrito.

Formato simples:

```markdown
## YYYY-MM-DD - titulo curto

Original:

-

Contexto, se lembrar:

-

Possivel destino:

-
```

## Regra De Processamento

Quando Thiago pedir para organizar o Inbox:

1. preservar o original;
2. separar ideias misturadas;
3. mover cada captura para o destino certo;
4. criar perguntas somente quando o destino nao puder ser inferido;
5. manter aqui apenas o que ainda estiver sem destino claro.
