# Clients and Mentoring

Use esta pasta para clientes, mentorias, diagnosticos e planos de acao.

Cada cliente deve ter:

- contexto;
- transcricoes;
- diagnostico estruturado;
- decisoes e proximos passos.

Quando for consultoria ou diagnostico operacional, tambem pode ter:

- oportunidades de IA e automacao;
- plano de acao.

Quando for mentoria, prefira:

- plano da mentoria;
- roteiro da proxima sessao;
- logs/feedback das sessoes.

Regra:

Nao misture cliente com conteudo de LinkedIn ou framework generico. Se algo aprendido com o cliente virar metodo reutilizavel, crie uma versao anonima em `07-Frameworks-and-Processes/`.
