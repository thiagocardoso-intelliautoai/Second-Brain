# AI Automation Project Documentation

## Objetivo

Padronizar documentacao de projetos de IA/automacao sem engessar a execucao.

## Template

### 1. Contexto

- Quem pediu?
- Qual area?
- Qual problema?
- Qual processo atual?

### 2. Dor

- O que e lento?
- O que e repetitivo?
- O que e esquecido?
- O que gera erro?
- Qual custo operacional?

### 3. Processo atual

Descrever As Is:

- inputs;
- atividades;
- ferramentas;
- pessoas;
- outputs;
- excecoes.

### 4. Oportunidade

- O que pode ser redesenhado?
- O que pode ser automatizado?
- Qual parte deve continuar humana?

### 5. Modo recomendado

Classificar:

- assistido;
- workflow;
- agent;
- hibrido.

### 6. Arquitetura

- ferramentas;
- dados;
- triggers;
- passos;
- checkpoints;
- outputs.

### 7. ROI

- tempo economizado;
- custo/hora ou custo mensal da pessoa/processo impactado;
- valor economizado em R$;
- qualidade;
- velocidade;
- receita;
- risco reduzido;
- custo de construir e manter.

Regra:

Todo projeto de IA/automacao deve ter uma equacao de valor registrada quando houver dados suficientes. No minimo:

```text
Tempo economizado + R$ economizado + reducao de esforco operacional
```

Para projetos internos, essa equacao prova valor, ajuda a priorizar proximas melhorias e protege a automacao de virar apenas uma demo tecnica.

Para ofertas futuras, essa equacao vira insumo comercial: mostra ganho concreto, reduz incerteza e explica por que a automacao merece investimento.

### 8. Plano de execucao

- fase 1;
- fase 2;
- fase 3;
- criterio de sucesso;
- riscos.

### 9. Handoff

O handoff deve permitir que outra IA ou pessoa continue sem perder contexto.

Incluir:

- objetivo;
- decisoes tomadas;
- arquivos relevantes;
- pendencias;
- proximo passo exato.

## Status operacional

Este framework continua como template de referencia, nao como skill pronta.

Use para documentar projetos reais de IA/automacao quando houver necessidade de clareza, handoff ou continuidade. So transformar em prompt, template mais rigido ou skill quando houver demanda recorrente por esse tipo de documentacao.
