## Regra rapida

- Skill global: metodo transversal, usado em mais de uma frente.
- Skill local: metodo preso a um projeto, cliente, repo ou pipeline.
- Mentor: lente operacional para decidir, criticar ou perguntar melhor.
- Framework: pensamento reutilizavel que ainda nao necessariamente virou ferramenta.
